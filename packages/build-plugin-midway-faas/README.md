# build-plugin-midway-faas

Plugin which make Rax App can be used in [Midway FaaS](https://www.yuque.com/midwayjs/faas)

## Usage

```json
{
  "plugins": [
    [
      "build-plugin-rax-app",
      {
        "targets": [
          "web"
        ]
      }
    ],
    "build-plugin-midway-faas"
  ]
}
```
